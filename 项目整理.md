### 本地运行建议

1. 准备 JDK 17、Maven、MySQL、Redis。
2. 初始化数据库：导入 `h/sql/mysql/jamyu.sql`，必要时参考 `youlai_boot.sql`。
3. 修改 `h/src/main/resources/application-prod.yml` 中的数据库、Redis、OSS 等配置。
4. 启动：

```bash
cd D:\Microsoft\work\zx-jy-all\h
mvn spring-boot:run
```

### 后端需要优先处理的问题

- `application-prod.yml` 中存在明文数据库密码、短信配置、AI API key 默认值、JWT secret 等敏感信息，建议改为环境变量或本地私有配置文件，并轮换已经暴露的凭据。
- `Dockerfile` 中 `ADD target/youlai-boot.jar app.jar` 与 Maven `finalName=jamyu` 不一致，实际打包产物大概率是 `target/jamyu.jar`。
- `Dockerfile` 暴露端口是 `8989`，但 `application-prod.yml` 配置为 `8990`，部署前需要统一。
- `application.yml` 默认激活 `prod`，本地开发建议增加 `dev` 配置，避免直接连接生产/公网资源。
- README 仍是模板内容，并且终端读取时中文有编码乱码，建议后续重写为项目真实说明。
